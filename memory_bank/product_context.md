# 产品背景
Daylo 是一个面向单用户的日计划工具，目标是把用户的 goals、tasks 和可用时间块组合成一个可执行的当天 schedule。

# 目标用户
- 需要把长期目标拆解成当日执行计划的个人用户
- 当前阶段先聚焦单用户，不处理团队协作、登录或多用户隔离

# MVP 核心价值
- 录入 goals 与 tasks
- 录入当天可用时间块
- 自动生成当天 schedule
- 为 schedule 提供可读的 reasoning
- 支持基于反馈重新规划

# 当前产品边界
- 只做日计划，不做周计划
- reasoning 先用规则生成，不依赖 AI
- re-planning 先做简单反馈驱动，不做复杂对话式 agent
